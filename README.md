# AI/ML Internship — Day 4 (Raritone)

## Overview
This project focuses on improving human pose detection and exploring advanced AI concepts such as body segmentation, virtual try-on systems, and clothing recommendation logic. The work combines computer vision techniques with practical implementation using MediaPipe and OpenCV.

---

## Tasks Completed

### Task 1 — Improve Pose Detection Accuracy
- Implemented pose detection using OpenCV and MediaPipe  
- Improved accuracy by tuning model parameters  
- Enabled smoothing for stable landmark tracking  
- Achieved better real-time performance and detection consistency  

---

### Task 2 — Virtual Try-On AI Models
- Studied how AI enables digital clothing trials  
- Understood pose detection and body mapping techniques  
- Learned how clothing is overlaid on detected body structures  
- Explored applications in e-commerce and augmented reality  

---

### Task 3 — Body Segmentation Methods
- Implemented segmentation using MediaPipe  
- Separated human body from background  
- Applied background blur for visual clarity  
- Demonstrated real-time body isolation  

---

### Task 4 — Clothing Recommendation Logic
- Designed simple logic for clothing size prediction  
- Used body measurements like shoulder width  
- Categorized sizes into Small, Medium, and Large  
- Explored its use in personalized fashion systems  

---

### Task 5 — MediaPipe vs OpenPose
- Compared two pose detection frameworks  

| Feature | MediaPipe | OpenPose |
|--------|----------|----------|
| Speed | Fast | Slower |
| Accuracy | Good | Very High |
| Setup | Easy | Complex |
| Resource Usage | Low | High |
